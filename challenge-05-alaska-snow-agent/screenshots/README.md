## Application Interface

The Streamlit application features:
- Alaska Department of Snow branding with snowflake icon
- Clean, light blue background theme
- Chat interface for user queries
- Virtual assistant responses powered by RAG and Gemini 2.5 Flash
- Support for snow removal questions and weather forecasts
